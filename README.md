Title: Maximizing the number of real solutions of a parametrized polynomial system: Dietmaier optimization and omnitropic descent

Abstract: 

Finding a parameter value for which the number of real solutions of a
parametrized polynomial system is maximized is a fundamental problem in
numerical algebraic geometry. In 1998, Peter Dietmaier achieved the goal
of finding a parameter for which all the possible solutions are real
for the famous Stewart-Gough platform system, by gradient descent along a
specific objective function which we call the Dietmaier function.



In this talk we will describe a modified version of the algorithm
inspired by Dietmaier's idea. This new version, which we call
omnitropic descent, attempts to consider different choices of directions
possible in the parameter space to track the solutions in than the specific
branch Dietmaier chose. By doing this computation in a lazy way, that is,
tracking a solution only when it is necessary, we can make each
iteration fast and might even benefit from being able to track the solution to
other discriminant chambers without crossing a singular point. If time
permits, we will also describe some concrete examples that we have tried
this on.
